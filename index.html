CS Bio

I started using a computer around when I was 4 or 5 years old. It was one of those ancient ThinkPads my parents used for work. I've used many types of computers, on most major operating systems. (Pocket PCs, phones, laptops, custom desktops, Raspberry Pi's, etc.)I use computers for productivity, gaming, messaging, and pretty much everything else on a daily basis. I think a computer scientist designs and builds features for software on a daily basis.

My introductory programming course I took in high school had a game design project using Java Swing/Applets, and I think the experience of building that terrible game was something that led me to study computer science. I'm taking this course because my friend told me to register 15 minutes before class started and because I thought web development might be a useful skill.

For front-end, some experience with UI/UX design and knowledge of HTML/CSS would be helpful in building attractive and user-friendly interfaces. For back-end, knowing a language like JS or Python and an understanding of how databases work would be useful. Since I'm planning on being a software engineer after graduating, I see myself using computers a lot more for coding and other work-related tasks in the future.
